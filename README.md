# Распайка микрофона
|MP34DT01         |Atom lite        |Примечание       | 
|-----------------|-----------------|-----------------|
|3V |3V ||
|GND |GPIO22 ||
|SEL |GPI23 |GND
|CLK |GPIO19 |GND R or 3.3V L|
|DAT |GPIO33 ||

1. Скопируте папку ledstrip в /esphome/my_components
2. Cкопируйите led-strip.yaml
